## 项目概述

### 🎯 项目介绍

CAD-FAI 是一个完全自动化的 Web 应用，用于从 CAD 图纸中自动提取尺寸标注并生成符合工程标准的 FAI（第一件检验）报告。

### ✨ 核心功能

| 功能 | 说明 | 状态 |
|-----|------|------|
| **自动识别尺寸** | 线性、直径(Ø/φ)、半径(R) | ✅ |
| **公差解析** | 对称(±) 和非对称(+/-) 公差 | ✅ |
| **ISO 2768-m** | 无公差时自动应用标准 | ✅ |
| **随机实测值** | 生成真实感的检验数据 | ✅ |
| **OK/NG 判定** | 自动评判合格/不合格 | ✅ |
| **多格式输出** | CSV、JSON、HTML | ✅ |
| **CAD 文件上传** | 支持 DXF 格式 | ✅ |
| **中文支持** | 完全中文界面和输出 | ✅ |
| **无服务器部署** | Cloudflare Workers | ✅ |

## 功能详解

### 1️⃣ 尺寸识别

#### 线性尺寸
```
格式: 50, 10 ±0.02, 15 +0.02/-0.01
识别: ✓ 完全支持
```

#### 直径标注
```
格式: Ø20 ±0.02, φ25, Ø15 +0.02/-0.01
识别: ✓ 完全支持
符号: Ø 或 φ 都可识别
```

#### 半径标注
```
格式: R5, R5 ±0.1
识别: ✓ 完全支持
```

### 2️⃣ 公差解析

#### 对称公差
```
输入:  ±0.02
输出:  { tolPlus: 0.02, tolMinus: 0.02 }
```

#### 非对称公差
```
输入:  +0.02/-0.01
输出:  { tolPlus: 0.02, tolMinus: 0.01 }
```

### 3️⃣ ISO 2768-m 标准

当标注**无显式公差**时自动应用：

```
范围           等级 m 公差
≤ 6mm          ±0.1
6-30mm         ±0.2
30-120mm       ±0.3
> 120mm        ±0.5
```

**示例**:
```
输入: 50 (无公差)
应用: ISO 2768-m
结果: 50 ±0.3 (30-120mm 范围)
```

### 4️⃣ 随机实测值生成

**算法特点**:
```
✓ 90% 概率: 在公差范围内 (OK)
✓ 10% 概率: 超出范围 (NG)
✓ 精度: 0.0001mm (四舍五入)
```

### 5️⃣ OK/NG 判定

**判定逻辑**:
```
OK:  measured ≥ (nominal - minus) AND measured ≤ (nominal + plus)
NG:  否则
```

### 6️⃣ 输出格式

#### CSV 格式 (推荐)

**特点**:
- ✅ 中文表头支持
- ✅ UTF-8 BOM 编码（Excel 自动识别）
- ✅ 完全兼容 Excel、Google Sheets
- ✅ 易于数据分析

**示例**:
```csv
"序号","标注类型","标称尺寸","公差(+/-)","实测值","判定"
"1","直径","20","+0.0200/-0.0200","20.0156","OK"
"2","半径","5","+0.1000/-0.1000","5.0234","OK"
"3","线性","10","+0.0200/-0.0100","9.9987","OK"
```

#### JSON 格式

**特点**:
- ✅ 结构化数据
- ✅ 易于编程处理
- ✅ 前端集成方便

#### HTML 格式

**特点**:
- ✅ 网页预览
- ✅ 美化显示（OK 绿色、NG 红色）
- ✅ 易于打印
- ✅ 实时查看

## 常见问题

### ❓ 使用相关

**Q: 怎样获取 DXF 文件？**

A: 在你的 CAD 软件中：
- AutoCAD: 文件 → 另存为 → 选择 DXF 格式
- LibreCAD: 文件 → 导出 → 选择 DXF 格式

**Q: 为什么显示"未找到标注信息"？**

A: 可能的原因：
1. DXF 文件中没有标注
2. 标注格式不被识别
3. DXF 文件损坏

**Q: 能处理多个文件吗？**

A: 目前一次处理一个文件。可逐个处理并合并结果。

**Q: CSV 文件如何在 Excel 中打开？**

A: 
1. 直接双击 CSV 文件（推荐）
2. 或在 Excel 中打开

### ⚙️ 配置相关

**Q: 如何修改 ISO 2768-m 默认值？**

A: 编辑 `src/parse.ts` 的 `iso2768mApprox()` 函数

**Q: 如何调整 OK/NG 的比例？**

A: 编辑 `src/parse.ts` 的 `generateMeasurement()` 函数中的概率值

**Q: 如何自定义 CSV 表头？**

A: 编辑 `src/excel.ts` 中的 `headers` 数组

### � 部署相关

**Q: 部署成本是多少？**

A: Cloudflare Workers 提供：
- 100 万次请求/天 免费额度
- 足以满足大多数用户需求

**Q: 如何查看部署日志？**

A: Cloudflare 仪表板 → Workers → 查看实时日志
